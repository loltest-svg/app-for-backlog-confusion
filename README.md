# App for Backlog Confusion 📚

An interactive web app to transparently show my academic backlog progress — built with honesty and love.

## Features

- 🎯 **Interactive Mind Map** — Hover over each subject to see detailed info floating in space
- 🌑 **Dark Space Theme** — Sleek dark UI with particle effects and glowing orbs
- 📊 **Progress Tracking** — Color-coded status for each backlog (Cleared, Giving, Planned, etc.)
- 💛 **Heartfelt Message** — Click the important button for a personal message
- 📱 **Responsive** — Works on desktop and mobile
